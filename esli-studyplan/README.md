# Esli Study Plan

Classroom chess openings trainer — curriculum packs, teacher/student roles, and learn-by-moving drills.

## Live site

Once GitHub Pages is enabled on this repository:

**https://eniola-giwa.github.io/esli-studyplan/**

## Local preview

```bash
cd esli-studyplan
python3 -m http.server 8080
# open http://127.0.0.1:8080/
```

## Features

- Search openings by name across curriculum packs and the full ECO book
- Beginner / Club / Advanced curriculum packs
- Teacher vs Student classroom roles
- Difficulty caps, hints, streaks, local progress
- Works fully static in the browser (no backend required)
